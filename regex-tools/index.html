JavaScript 正则表达式工具

常用正则表达式

邮箱地址
/^[^\s@]+@[^\s@]+\.[^\s@]+$/
匹配标准的邮箱地址格式
手机号码 (中国)
/^1[3-9]\d{9}$/
匹配中国大陆11位手机号码
URL地址
/^https?:\/\/(www\.)?[-a-zA-Z0-9@:%._\+~#=]{1,256}\.[a-zA-Z0-9()]{1,6}\b([-a-zA-Z0-9()@:%_\+.~#?&//=]*)$/
匹配HTTP/HTTPS协议的URL
身份证号 (中国)
/^[1-9]\d{5}(18|19|20)\d{2}((0[1-9])|(1[0-2]))(([0-2][1-9])|10|20|30|31)\d{3}[0-9Xx]$/
匹配18位中国大陆身份证号码
IPv4地址
/^((25[0-5]|(2[0-4]|1\d|[1-9]|)\d)\.?\b){4}$/
匹配IPv4地址格式
日期格式 (YYYY-MM-DD)
/^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$/
匹配YYYY-MM-DD格式的日期
密码强度 (至少8位，包含字母和数字)
/^(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{8,}$/
至少8位，必须包含字母和数字
中文字符
/[\u4e00-\u9fa5]/
匹配中文字符
